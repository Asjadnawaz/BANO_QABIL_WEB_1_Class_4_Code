# 🌐 Web Development with AI – Class 4

This repository contains my Class 4 practice work from the Web Development with AI course.

In this session, I learned how to build complete multi-page websites with proper navigation and user input features.

## 📚 What I Learned
- Creating navigation bars using `<nav>`
- Building multi-page websites using anchor links
- Using HTML forms (`<form>`) for user input
- Using attributes like `placeholder`
- Connecting multiple pages (Home, Contact, etc.)

## 🎯 Practice Work
- Built a multi-page website on a chosen topic
- Created consistent navbar across all pages
- Linked pages using anchor tags
- Designed a Contact Us form
- Added relevant content for better structure

## 🚀 Goal
To understand how real websites are structured using multiple connected HTML pages, navigation, and forms.
